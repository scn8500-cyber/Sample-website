Analytics Playground

A tiny site to learn GA4 + GTM with real data. Open your GA Realtime tab, then click stuff below.

1) Custom events via dataLayer

These buttons push custom events. In GTM, create a Custom Event trigger named button_click and fire a GA4 Event tag.

2) Outbound link

Enhanced Measurement in GA4 can auto-capture outbound clicks. Or do it via GTM link-click triggers.

Visit example.com

3) Simple form

Submitting this form pushes a sign_up event.

After submit, check GA Realtime → Events.

Debug

Open DevTools → Console to see events as they’re pushed to dataLayer.

Replace GTM-XXXXXXX with your GTM container ID. Then configure GA4 tags in GTM as described in the README.
